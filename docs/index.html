Architecture

Five-Layer Statistical Architecture

Each layer builds on the one below. Composites exist only at the top, computed at query time, never stored as truth.

1

Raw Indicators

Stored in PostgreSQL + PostGIS. Every value carries its GEOID, source metadata, vintage year, and a CV reliability flag (Census Bureau thresholds: high/moderate/low).

2

Validated Features

ICE (Krieger 2016), Dissimilarity Index (Massey & Denton 1988), Isolation Index (Lieberson 1981), Housing Cost Burden (HUD). Literature-grounded, not invented.

3

Factor Scores

Exploratory Factor Analysis with oblimin rotation. Named by loading profile (e.g., "Economic Distress," "Housing Burden"), not by number. Expect 5-8 factors from 50+ indicators.

4

Spatial Analysis

LISA cluster maps, Geographically Weighted Regression, multilevel variance partitioning (tract / county / state), SKATER regionalization.

5

Composite Views (Query-Time Only)

Users can compute custom composites via the API, but the system never imposes them. Geometric mean aggregation with sensitivity analysis under ±20% weight perturbation.

Methods

Research-Grounded Statistical Methods

Every method traces to published validation. No unvalidated composites.

ICE (Index of Concentration at the Extremes)

Measures spatial polarization between privileged and deprived populations without collapsing race and income into a dimensionless score.

Krieger et al. 2016

Dissimilarity Index

D = 0.5 × Σ|bi/B - wi/W|. The canonical measure of residential segregation, validated across 35+ years of research.

Massey & Denton 1988

Exploratory Factor Analysis

EFA with oblimin rotation discovers which indicators belong together. Kolak et al. found 4 SDOH factors explaining 71% variance at 72K tracts.

Kolak et al. 2020

OLS Regression

Ordinary least squares with standard errors, t-statistics, and p-values. Implemented in pure Go with Gauss-Jordan matrix inversion.

Pure Go

Blinder-Oaxaca Decomposition

Generalized two-group decomposition for equity gap analysis. Separates explained vs. unexplained components of outcome disparities.

Pure Go

Threshold Detection

Piecewise linear breakpoint detection identifies where outcomes change. Policy-interpretable: "above 35% poverty, diabetes prevalence increases 4x faster."

Segmented regression

Origin

From Madison to National

PDI began as the Madison Equity Atlas — a 22-layer GIS platform analyzing 125 census tracts in Dane County, WI for Common Wealth Development.

The Atlas produced Five Mornings in Madison (a narrative document profiling five households shaped by structural conditions), a founding partnership proposal for a 12-organization coalition, and 70 evidence cards mapping policy positions to tract-level data.

PDI generalizes the Atlas methodology to national scale. The same pipeline that told the story of five Madison families can now tell the story of any county in America, grounded in that county's actual tract-level data.

Get Started

Quick Start

# Start PostgreSQL + PostGIS
docker compose up -d

# Run migrations
go run ./cmd/pdi migrate up

# Fetch Dane County, WI data
go run ./cmd/pdi fetch --state 55 --county 025 --year 2023

# Run the full pipeline
go run ./cmd/pdi pipeline run --state 55 --county 025

# Start the API server
go run ./cmd/pdi serve --port 8340
